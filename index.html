Пузырьковая навигация для сайта

В данном уроке мы создадим пузырьковую навигацию с помощью jQuery. Идея заключается в том, чтобы сделать несколько круглых иконок, на которых при наведении курсора мыши "раскрывается" круглое подменю с дополнительными пунктами. Мы будем использовать плагин jQuery Easing для получения замечательного эффекта сглаживания.

Урок Демонстрация

Коллапсирующая навигация по сайту

В данном уроке мы создадим коллапсирующее меню, которое содержит вертикальные навигационные колонки и выскальзывающую область с контентом. Когда курсор мыши проходит над пунктом меню сверху выскальзывает изображение, а снизу - подменю. При нажатии на пункт подменю все меню сворачивается, как карточная колода, и соответствующий контент выскальзывает в освободившееся пространство.
